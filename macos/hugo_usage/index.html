markdownでウェブページを作成できる「hugo」の使い方

hugoメモ

hugoの「Quick Start」のページを読んでインストールする。

  • Quick Start |Hugo

トップページはconfig.tomlを編集して作る。

フッターの編集は、themesフォルダ内の、使用しているテーマ(たとえば「accessible-minimalism」)のフォルダの中の: layouts/partials/footer.html

を編集する。

ターミナルで

hugo server

と入力すると

http://localhost:1313

で作成しているページを開くことができる。

markdownファイルを編集したら、コマンド-rでリロードすると変更が反映される。
