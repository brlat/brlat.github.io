VoiceOverで便利に使えるMac OSアプリ

バーチャル音声ケーブルアプリ、Rogue Amoeba社の「Loopback」

109ドルの有料アプリ。

一つのアプリから出力される音声を別のアプリの入力につなぐことができるアプリ。

オンラインで流れる音声を録音したりできるようになる。

  • Rogue Amoeba | Loopback: Cable-Free Audio Routing

音声編集アプリ、「Amadeus Pro」

60ドルくらいの有料アプリ。

  • Amadeus Pro - Audio waveform editor / sound and voice recorder for macOS X

mp3タグ編集アプリ「Mp3tag」

Windows版は無料ですが、Mac版は有料20ドルくらい。

フォルダ名やファイル名からmp3タグに変換、その逆ができます。

Windowsではフォルダの区切りは円マークですが、Macではスラッシュになります。

またMacでは.wavファイルにつけたタグは有効にならないようです。

.aifファイルのタグは有効になっています。

  • Mp3tag: the universal Tag Editor for Mac

メディアプレイヤー「VLC」

Mac版ではVoiceOverの操作で、局の始まりと終わりの時間を指定して、局の一部だけを連続再生することができます。(Windows NVDAではうまく、私にはできませんでした

  • Official Download of VLC media player for Mac OS X - VideoLAN

音楽・音声編集ソフトDAW「Reaper」

60ドルの有料アプリ。

音声編集とソフトウェア音源を使った音楽編集ができます。

Mac用のOsaraをインストールしてVoiceOverでも使用できます。

コンテキストメニューを出すには、コントロール-1、コントロール-2、コントロール-3を押します。(Windowsではアプリケーションキー)

  • REAPER | Audio Production Without Limits

画面の文字認識「VOCR」

NVDAの、NVDAキー-Rのように、そのままでは音声読み上げできない画面上の文字をOCR認識して読み上げるMac VoiceOver用のアプリ。

認識できるのは英語だけ。

  • VOCR chigkim/VOCR - GitHub

有料(サブスクリプション)の「FishScreenReader」というのもある。

テキストエディタ「CotEditor」

オープンソースで無料で使用できるアプリ。

  • CotEditor - The Plain-Text Editor for macOS

テキストエディタは他にも、「Tincta」や「TextMate」もVoiceOverで使用できる。

ftpアプリ「Cyberduck」

Macで使いやすいftpソフトをさがして、いくつか試しましたが今のところ、Cyberduckが使いやすそうでした。

  • Cyberduck | Libre server and cloud storage browser

「辞書 by 物書堂」

iOS版で購入した辞書がそのままMacでも使えます。

  • macOS版「辞書 by 物書堂」をリリース
